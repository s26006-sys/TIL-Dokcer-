# hibernate란?
hibernate는 자바 언어를 위한 ORM 프레임워크이다.
(ORM은 객체와 관계형 데이터 베이스를 연결해주는것)
# ddl-auto
ddl-auto는 하이버네이트가 제공 하는 다양한 기능 중에서 엔티티만 등록해 놓으면 DDL을 자동으로 작성하여 테이블을 생성하거나 수정해주는 설정이다.

# ddl-duto 속성의 종류

|종류|설명|
|----|----|
|create|엔티티로 등록된 클래스와 매핑되는 테이블을 자동 생성|
|create-drop|create와 비슷하지만 애플리케이션이 종료될 때 테이블을 삭제한다는 차이가 있다.|
|update|테이블을 생성하는데까지는 create와 동일하지만 기존테이블이 존재하면 테이블의 컬럼을 변경한다.|
|valodate|
|none(default)|

- 추가로 create 과정에서 해당 클래스와 매핑되는 테이블이 존재하면 기존 테이블을 삭제(drop)하고 테이블을 생성한다.
- update는 모든 변경사항을 반영하지 않는다.
